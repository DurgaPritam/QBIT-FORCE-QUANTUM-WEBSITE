# Qbit Force Quantum

Marketing site (React + Vite) with a Spring Boot CMS backend.

## Quick start

**Frontend**

```bash
npm install
# Create .env with: VITE_API_BASE_URL=http://localhost:8080/api
npm run dev
```

**Backend** — see [backend/README.md](backend/README.md) for MySQL, JWT, and admin setup.

**Admin panel:** `http://localhost:5173/qbitadmin-2026-login`
